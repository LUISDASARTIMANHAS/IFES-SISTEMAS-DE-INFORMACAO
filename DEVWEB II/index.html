    • Desenvolvimento do trabalho prático: Entrega 0.

      A Entrega 0 do trabalho prático se refere ao desenvolvimento de uma pequena parte do Projeto Locadora sem que sejam utilizados frameworks web como, por exemplo, Angular ou Spring Boot. Será utilizado apenas as tecnologias fundamentais da plataforma J2EE como servlet e JSP, além de tecnologias de front end como HTML, CSS e JavaScript.

      Para a Entrega 0 deve ser desenvolvido o caso de uso Cadastrar Ator. Conforme a especificação de análise disponibilizada, o caso de uso Cadastrar Ator possui os seguintes eventos de casos de uso: incluir novo ator, alterar dados de ator, excluir ator e consultar dados de ator. Esses eventos de caso de uso devem ser implementados.

      Ademais, para a Entrega 0 deve-se utilizar a arquitetura de camadas do modelo MVC apresentado em sala.

      Bom trabalho.
